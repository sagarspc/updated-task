IEC Registration



What is IEC Code

IEC code is necessary for import/export of goods. The first requirement before you start an import/ export business in India is to obtain an IEC. IEC Code is unique 10 digit code issued by DGFT – Director General of Foreign Trade, Ministry of Commerce, Government of India. An IEC Number allotted to an applicant shall be valid for all its branches/divisions/units/factories as indicated on the IEC number.

However, under the following circumstances IEC code is not required:

1. Import/export of goods for personal use, which is not connected with trade, manufacture or agriculture.

2. Import/export by government ministries and departments, and certain notified charitable organizations.

1. What is IEC Code?

It stands for Import Export Code. It is required for person importing or exporting good and services to or from India.

2. Who should obtain IEC Code?

Any person who wishes to import or export any goods and services to or from India shall obtain IEC Code.

3. Can individuals obtain IEC Code?

Yes, individuals can obtain IEC Code.

Step 1: Applicant can apply for online Registration at DGFT Site(http://dgft.gov.in)

Step 2: Click on Importer Exporter Code (IEC) and select Online IEC Application.

Step 3: Login ID by entering PAN details of the Entity.

Step 4: Complete all details by filling relevant information online like:

• IEC Master

• Branch

• Director

• Pay Fee through any designated bank online

• Attach Documents

Step 5: After the requisite procedure, the applicant will sign the Application and submit the same

Step 6: After the whole submission, a Customer Reference Number is generated.

• Copy of PAN Card

• Bank certificate or cancelled cheque bearing the entire name.

• Address Proof

• Incorporation Proof

F.A.Q

  • 1. Can a person/ entity have more than 1 IEC?

    No, a person cannot have more than 1 IEC.

  • 2. What to do if my IEC code is lost?

    I need to provide an affidavit so that a copy can be issued.

  • 3. Can I surrender my IEC Code?

    Yes, you can surrender your IEC code. You need to inform the issuing the authority about such decision. And they shall immediately cancel your IEC number after electronically transmitting it to DGFT for onward transmission to the Customs and Regional Authorities.

  • 4. Who should get IEC number?

    Any person who intends to start his/her import/export business shall get IEC number.

  • 5. Can individuals apply for IEC Code?

    Yes, individuals can apply for IEC code.

  • 6. How long does it take to get IEC code?

    Normally, it takes 5 to 10 days to get IEC code.

  • 7. Do we need to show any proof while submitting application for obtaining IEC Code?

    No, we don’t need to show any proof of any import or export to obtain IEC Code.

  • 8. Can I modify my IEC Code?

    Yes, you can apply for modification.

  • 9. What is the procedure for modification of IEC Code?

    It requires re-submitting the application again to DGFT along with the required documentation and fees.

  • 10. What is the procedure for modification of IEC Code?

    It requires re-submitting the application again to DGFT along with the required documentation and fees.

  • 11. Are there any returns to be filed?

    No, you are not required to file any return.

  • 12. Are different IEC required for different items?

    No, once an application is done, it shall be valid for all items.

  • 13. Does IEC code have lifetime validity?

    Yes, IEC code has lifetime validity.
